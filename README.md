# 📄 RAG Chatbot using LangChain + FAISS

## 🚀 Project Overview

This project is an **AI-powered chatbot** that can answer questions from custom PDF documents using **Retrieval-Augmented Generation (RAG)**.

Instead of relying only on a language model, this system retrieves relevant information from documents and provides **accurate, context-aware answers**.

---

## 🧠 Key Features

* 📄 Load and process PDF documents
* ✂️ Smart text chunking using recursive splitting
* 🔎 Semantic search using vector embeddings
* ⚡ Fast retrieval using FAISS (Vector Database)
* 🤖 Question answering with LLM (OpenAI GPT)
* 🔁 Interactive chat loop

---

## 🏗️ Tech Stack

* **Python**
* **LangChain**
* **FAISS (Facebook AI Similarity Search)**
* **Sentence Transformers**
* **OpenAI GPT (LLM)**
* **PyPDF Loader**

---

## 📂 Project Structure

```
RAG-Chatbot/
│── main.py
│── requirements.txt
│── README.md
│── sample.pdf
```

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/rag-chatbot.git
cd rag-chatbot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

Or manually:

```bash
pip install langchain langchain-community faiss-cpu sentence-transformers openai pypdf
```

---

## 🔑 Setup API Key

Set your OpenAI API key:

```bash
export OPENAI_API_KEY="your_api_key_here"   # Mac/Linux
```

```bash
set OPENAI_API_KEY=your_api_key_here        # Windows
```

Or inside Python:

```python
import os
os.environ["OPENAI_API_KEY"] = "your_api_key_here"
```

---

## ▶️ How to Run

```bash
python main.py
```

Then ask questions in the terminal:

```
You: What is this document about?
Bot: This document explains...
```

---

## 🔄 Workflow

1. Load PDF documents
2. Split text into chunks
3. Convert text into embeddings
4. Store embeddings in FAISS
5. Retrieve relevant chunks based on query
6. Generate answer using LLM

---

## 📊 Example Use Cases

* 📚 Study material Q&A
* 📑 Resume or document analysis
* 🏢 Company policy assistant
* 📖 Research paper summarization

---

## 📈 Future Improvements

* 🌐 Streamlit / Gradio UI
* 📂 Multiple document support
* 💬 Chat memory
* 🧠 Local LLM (no API cost)
* ☁️ Deployment (Hugging Face / Render)
